# vak_spisok
Список журналов из перечня ВАК. 

https://maximtimof.github.io/vak_spisok/

## ДЛЯ УДОБСТВА ПОИСКА
Каждый российский аспирант должен публиковать свои научные статьи журналах из Перечня ВАК, соответствующие шифру
его специальности. Официальный список постоянно обновляется и публикуется в формате огромном PDF, и он не удобен для чтения.
Поэтому PDF сконвертировал в Excel, потом обработал его в R Studio используя магию пакета "tidyverse",
и далее загрузил в дашборд Яндекс Даталенс для быстрой онлайн фильтрации.
